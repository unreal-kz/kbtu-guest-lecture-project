# Lecture Objectives

Working title:

**ML-enabled Information Systems and Evaluation Metrics: From Problem Framing to Project Reuse**

## Audience

Bachelor students in an Information Systems context.

## Objectives

By the end of the lecture, students should be able to:

1. explain what makes an information system ML-enabled;
2. distinguish the real-world problem from the ML task formulation;
3. choose evaluation metrics that match the task and the decision context;
4. explain why model metrics alone do not fully evaluate an ML-enabled IS;
5. apply a simple evaluation framework to their own course or project work.

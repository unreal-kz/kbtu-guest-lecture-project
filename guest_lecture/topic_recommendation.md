# Guest Lecture Topic Recommendation

## Recommendation grounded in the current repository

### Primary topic
**ML-enabled IS & Evaluation Metrics**

This is the cleanest and strongest recommendation because it is explicitly written in the syllabus page reconstructed inside the repo.

## Why this topic is the best fit

It satisfies all currently known constraints:

- it is directly supported by the repository;
- it is aligned with the Information Systems course context;
- it sits in the lecturer's machine learning expertise zone;
- it can be made practical enough for bachelor students to reuse in their own projects.

## How to make the topic stronger without leaving the repo evidence base

The lecture should not be only a list of metrics.

Instead, it should be built as:

**From problem framing and IS requirements to ML evaluation and decision-making**

This is still repo-grounded, because the nearby syllabus topics explicitly include:

- Problem Framing theories in Information Systems
- IS Requirements & Solution Design
- Integrating Field Data for IS Decisions
- Professional Reporting & Stakeholder Presentation

## Recommended final lecture title

**ML-enabled Information Systems and Evaluation Metrics: From Problem Framing to Project Reuse**
